사용 방법

1

왼쪽 원문에 글을 입력하면 실시간으로 글자 수가 계산됩니다.

2

원문 영역 상단의 'HTML 태그 제거' 버튼을 누르면 원문 내 불필요한 HTML 태그가 제거됩니다.

3

'앞뒤 문자 붙이기' 버튼을 클릭하면 설정에서 지정한 문자열이 원문의 각 줄에 앞뒤로 추가됩니다.

4

'클립보드 복사' 버튼을 누르면 변환된 결과가 클립보드에 복사됩니다. 이를 블로그 HTML 편집기에 붙여넣으면 됩니다.
